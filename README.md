## Project Structure

##  Features

- **Navigation Bar**
  Fixed navbar with smooth scroll to sections.

- **Hero Section**
  Hero image with fade edges and animated wave-hover buttons.

- **About Me**
  Profile picture with responsive text about my journey and background.

- **Skills & Education**
  Cards that scroll horizontally, each linking to a separate page.

- **Leisure Time / Hobbies**
  Hobby cards (Swordplay, Snooker, Target Practice, Ancient Insights, Relax & Recharge) with clickable links.

- **Footer**
  Social icons (Instagram, Facebook, LinkedIn, GitHub, Twitter) with hover effect.

---

##  Responsiveness

- Desktop: Full layout with horizontal card scrolling.
- Tablet/iPad: Same layout adapted to screen width.
- Mobile:
- Profile picture stacked above "About Me" text.
- Cards remain scrollable horizontally.
- Fully stacked layout for readability.
